Ircbot

mernisse dev branch of ircbot

A small IRC robot in Python based on Twisted Words. The default modules currently support the following commands:

Brain Commands

Get/Set/Delete settings used by other plugins

  • botname: set variable value
  • botname: get variable
  • botname: del variable
  • botname: list

ASCII Art

The bot can emit ASCII art to the channel or provide a list of available art.

  • botname: ascii [art]
  • botname: ascii list

Impersonation Commands

Have the bot pull a quote from various people out of its brain

  • botname: be [name]

There are a lot of names that are valid, try archer!

Sending SMS

This can take advantage of the 'mobile' variable stored in the Brain, so if you want to get SMS messages from the bot make sure you ''set mobile [yourphonenumber]''. The format '+12125551234' is suggested.

  • botname: sms [nickname or phone number] [message]

Getting Weather

  • botname: weather [zip code]

If you set the 'wxlocation' variable in the Brain the parameter to this is optional and your stored location is used.

URL Tracking

Urls sent to the channel are caught, tracked and shortened for easy copy/pasting.

Surprises!

There are a few other easter eggs, using the bot is the best way to find out what fun could be hidden!
