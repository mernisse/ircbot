A small IRC robot in Python based on Twisted Words. The default modules currently support the following commands:

Brain Commands

Get/Set/Delete settings used by other plugins

  • botname: set variable value
  • botname: get variable
  • botname: del variable

Impersonation Commands

Have the bot pull a quote from various people out of it's brain

  • botname: be

There are a lot of names that are valid, try archer!

Debian Bugs

Saying 'debbugs ' anywhere in a channel with the bot will cause it to emit information on that bug.

Sending SMS

This can take advantage of the 'mobile' variable stored in the Brain, so if you want to get SMS messages from the bot make sure you ''set mobile ''. The format '+12125551234' is suggested.

  • botname: sms

URL Tracking

Urls sent to the channel are caught, tracked and shortened for easy copy/pasting.

Surprises!

There are a few other easter eggs, using the bot is the best way to find out what fun could be hidden!
